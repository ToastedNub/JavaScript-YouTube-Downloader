# JavaScript-YouTube-Downloader
 - Downloads YouTube Videos as MP3 or MP4, with no Quality Loss
 - This is Required: https://nodejs.org/en

 --- HOW IT WORKS ---


 - Requires NodeJS: https://nodejs.org/en/download/current

 - Run the "Start" file

 - Follow Instructions in console

 - Finished downloads will be in DOWNLOADS > MP3 / MP4




 --- HOW TO FIX ERRORS ---


 - If something is restricted in any way, the script may fail (you need a cookies file to fix this)

 - Download the Get Cookies.txt LOCALLY extension in Microsoft Edge: https://chromewebstore.google.com/detail/get-cookiestxt-locally/cclelndahbckbenkjhflpdbgdldlbecc

 - Sign into any YouTube Account set as over 18, then use the extension to export the txt

 - Rename the txt to "cookies.txt" and put it into the ASSETS folder

 - Cookies do expire, so this will need to be done again in the far future
